**Цель:**  
Изучить применение модели логистической регрессии и метода опорных векторов в задаче бинарной классификации.

**Описание задания:**  
Необходимо решить задачу классификации физических лиц по уровню дохода. Данные находятся в файле `adult.csv` (можно найти в материалах к занятию или скачать по ссылке).

Целевая переменная – `income`, принимает два значения:

- `<=50K`
- `>50K`

Остальные признаки описывают персональную информацию: возраст, образование, семейное положение и т.д. Подробное описание признаков можно получить с помощью функций Python (`describe`, `unique`) или по ссылке.  
Альтернативный источник данных: [archive.ics.uci.edu...et/2/adult](https://archive.ics.uci.edu/dataset/2/adult).

---

**Этапы работы:**

1. **Загрузка данных**  
   Загрузить данные в рабочую среду (Jupyter Notebook или аналогичную).

2. **Первичный анализ данных**  
   a) Проверить данные на пропуски. Удалить при обнаружении.  
   _Предложить альтернативный способ обработки пропусков._  
   b) Построить 1–2 графика на выбор. Графики должны быть информативными, отражать особенности датасета/класса/признака.  
   c) Преобразовать категориальные признаки.

3. **Разделение выборки**  
   Разделить данные на обучающую (80%) и тестовую (20%) части.

4. **Обучение моделей**  
   Обучить модели:
   - Логистической регрессии
   - Опорных векторов  
     на обучающей выборке.

5. **Оценка моделей**  
   Для тестовой выборки предсказать уровень дохода, сравнить с истинными значениями, посчитать точность с помощью функции `score`.

6. **Выводы**  
   a) Кратко описать выполненные преобразования данных.  
   b) Сравнить точность двух моделей.  
   c) Высказать мнение, насколько хорошо модели справились с задачей.  
   _Что можно сделать для улучшения результата?_
